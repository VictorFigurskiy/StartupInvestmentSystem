О проекте

Ознакомиться с сервисом можно просмотрев данное видео:)

Данный сервис позиционирует себя как Ваш помощник в поиске инвесторов для собственных проектов либо же предоставляет мощный функционал в поиске стартапов, которые соответствуют вашим критериям поиска для инвестиций. Еще никогда инвестировать не было так просто!

Над разработкой данного портала работала наша дружная команда.

Технологии, которые использовались в проекте:

  • Frameworks - Spring, Spring MVC, Spring Security;

  • ORM - Hibernate;

  • Database - MySQL;

  • WEB - JSP, JSTL, BootStrap, HTML, CSS, JavaScript;

  • Logger - Log4j;

  • Tests - Junit, Mockito;

  • Development tools - Maven, Git, GitHub, Tomcat, Jetty;

  • Cloud platform - Heroku.
